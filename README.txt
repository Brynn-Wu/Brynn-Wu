Ctrl + L 选中整行

Ctrl+S — 保存当前文件
Ctrl+P — 快速打开文件（按文件名搜索）
Ctrl+Shift+P — 命令面板（执行任意命令）
Ctrl+` — 切换集成终端（打开/关闭）
Ctrl+Shift+` — 新建终端

Ctrl+F — 当前文件查找
Ctrl+H — 当前文件替换
Ctrl+Shift+F — 全局查找（在工作区中）
Ctrl+Shift+H — 全局替换（在工作区中）

Ctrl+G — 跳转到指定行号
Ctrl+Shift+O — 跳转到当前文件的符号（函数/标签）
F12 — 转到定义
Ctrl+.（Ctrl+点）— 快速修复 / 快速操作（Quick Fix）
F2 — 重命名符号
Alt+Click 或 Ctrl+Alt+Down/Up — 多光标编辑（插入多个光标）
Shift+Alt+F — 格式化文档（代码格式化）
Tab（在 HTML/Emmet 上下文） — Emmet 展开缩写（速写 HTML）

Ctrl+/ — 切换行注释
Shift+Alt+A — 切换块注释
Alt+Up / Alt+Down — 上移/下移当前行或选中行
Shift+Alt+Down — 复制当前行到下一行（行复制）
Ctrl+Shift+B — 运行任务 / 构建（常用于编译/打包脚本）
Ctrl+B — 切换侧边栏可见性（增加编辑空间）
Ctrl+\ — 垂直拆分编辑器
Ctrl+Tab — 在打开的编辑器之间切换（快速导航）
Ctrl+K Ctrl+S — 打开键盘快捷键设置（查看/修改快捷键）
Ctrl+, — 打开设置（搜索并修改用户/工作区设置）
Alt+Z — 切换自动换行（在长行 HTML/CSS 时常用）
Ctrl+Shift+X — 打开扩展视图（安装 Live Server、Prettier 等）
Ctrl+Shift+G — 打开源代码管理（Git）面板
